# Log Analyzer

Security log analysis tool.

Alat analisis log keamanan.


## Description

Parses and analyzes security logs to detect suspicious patterns and potential intrusions.

## Deskripsi

Mem-parsing dan menganalisis log keamanan untuk mendeteksi pola mencurigakan dan potensi intrusi.


## Installation

- Navigate to the project folder
- No dependencies required

## Instalasi

- Masuk ke folder proyek
- Tidak perlu dependensi


## Usage

- Analyze: python log_analyzer.py /var/log/auth.log
- Demo: python log_analyzer.py --demo

## Penggunaan

- Analisis: python log_analyzer.py /var/log/auth.log
- Demo: python log_analyzer.py --demo
